Getting Started

What is CTFBuddy?

CTFBuddy offers a new way to play and learn penetration testing. Each game you play will increase your skills in areas that are needed within the pen-testing industry. You will use popular tools to complete challenges in the following categories:

  • Reverse Engineering
  • Cryptography
  • Web Security
  • Linux
  • Windows
  • Forensics

How do I setup my hacking lab?

To play security games on CTFBuddy, you will be required to have your own 'Attack Machine'. We recommend using the most popular operating system among hackers, Kali Linux. 

Step 1 - Installing VirtualBox

You may run Kali Linux as your main operating system, but we highly suggest you run it as a virtual machine. To do this, You can download the latest version of VirtualBox from:
https://www.virtualbox.org/
  

Step 2 - Installing Kali Linux

Once you have installed VirtualBox, you will need to download the Kali Linux operating system:
https://kali.download/virtual-images/kali-2022.1/kali-linux-2022.1-virtualbox-amd64.ova

Step 3 - Configuring Kali Linux

You can follow this Official Guide on running Kali Linux within the VirtualBox software:   https://www.kali.org/docs/virtualization/install-virtualbox-guest-vm/  

Step 4 - Connecting to CTFBuddy Virtual Machines

Victim Virtual Machines that are downloaded from CTFBuddy are FULL of security issues (yay), hence they should NEVER be connected to the internet. To keep the machines isolated, you will need to configure a new VirtualBox network adapter.

In VirtualBox, navigate to File -> select "Host Network Manager".

Once in the Host Network Manager, you will need to change the IPv4 Address to 192.168.2.2 and click "Apply". If you know what you are doing, you may use a different valid IP Address.

Finally, change the adapter name for your attack box and for each new CTFBuddy Victim Machine you download. To do this, navigate to the virtual machines settings and select the "Host-only Adapter" you just created.

And that's it, you're set! You shouldn't need to do this configuration again, but you will always need to make sure both your attacker and CTFBuddy Victim Machines are set to the "Host-only Adapter".

How do I play?

Step 1 - Select Game

You can head to game select and choose the game you want to play. Once their, click the "Download" button to download the vulnerable virtual machine, followed by the "Play" button.

Step 2 - Setting Up the Game

Once you have downloaded the game file and have clicked "Play", you will need to make sure that both your attack box and victim box are configured correctly, don't forget to set both machines to the custom Host-Only Adapter you created. You can follow the same guide you used for setting up Kali Linux to setup the Victim Machine. If you don't know how to do this, look at the "How do I setup my hacking lab?" part of this tutorial.

Step 3 - Playing the Game

Read the game instruction to get an idea of how to start. If you think you have found a flag, insert it into the flag box and click "Submit".

Step 4 - Earning and Using Points

Points are a vital part of CTFBuddy and are used to calculate your place on the leader boards. If you get stuck, you can also use your points to purchase hints or a tutorial video. To earn more points, you will need to submit more flags, or you can even earn 1000 points for creating your own game!
